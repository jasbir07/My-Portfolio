# My Portfolio

A modern developer dashboard built with Next.js featuring blogs, projects, and real-time GitHub/Spotify integration.

## ✨ Features

- Developer dashboard with activity feed
- Blog system with dynamic routes
- Project showcase
- GitHub & Spotify integration
- Dark mode support
- Admin dashboard
- Fully responsive

## 🛠️ Tech Stack

- **Next.js 14+** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Radix UI** - Components
- **Framer Motion** - Animations
- **SWR** - Data fetching

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📁 Structure

```
my-portfolio/
├── app/              # Pages & API routes
├── components/       # React components
├── data/            # blogs.json, projects.json
├── hooks/           # Custom hooks
├── lib/             # Utilities
└── styles/          # Global styles
```

## 🔧 Scripts

```bash
npm run dev      # Development server
npm run build    # Production build
npm start        # Start production
npm run lint     # Run ESLint
```

## 📝 Pages

- `/` - Home/Dashboard
- `/blog` - Blog listing
- `/projects` - Projects showcase
- `/contact` - Contact page
- `/labs` - Experiments
- `/now` - What you're doing
- `/admin` - Admin panel

## 📝 Customize

- **Blog posts** - Edit `/data/blogs.json`
- **Projects** - Edit `/data/projects.json`
- **Status** - Edit `/data/status.json`

## 🚀 Deploy

Push to GitHub and deploy on [Vercel](https://vercel.com) - Next.js will auto-detect.
